Que es una pagina web

Una página web es un documento individual en línea que contiene información y se puede ver a través de un navegador web. A diferencia de un sitio web, que es un conjunto de páginas relacionadas bajo un mismo dominio, una página web es solo una parte de ese todo. Estas páginas pueden incluir texto, imágenes, videos y otros elementos multimedia, y son creadas usando lenguajes de programación como HTML, CSS y JavaScrip

Caracteristicas de una pagina web

Es un archivo con su propia URL única (dirección en internet), almacena información diversa, como texto, imágenes, videos, enlaces y más, Se construye con lenguajes como HTML para la estructura, CSS para la presentación y JavaScript para la interactividad, se accede a ella a través de un navegador web (como Chrome, Safari o Firefox) que interpreta el código para mostrar el contenido y Puede servir para muchos propósitos, como mostrar información de un negocio, un blog, una tienda en línea, etc..
