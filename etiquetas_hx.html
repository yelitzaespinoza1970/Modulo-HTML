Tambien se puede utilizar la h2 para escribir unos parrafos de texto, no se recomienda pero se puede utilizar para escribir como si fuera en parrafos, aunque no es lo correcto.

Igualmente se puede utilizar la etiqueta h3 para escribir parrafos cortos, pero no es recomendable, ya que para escribir parrafos debe utilizarse la etiqueta p.
